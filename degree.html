Degree and Programm

COM1001 : Introduction to software Engineering

Is a module teaches the main ideas behind Software Engineering that are required for developing basic functional software that can solve a problem , the module covers the the process of building such systems, from the analysis of requirements to the implementation and execution, this module also develops teamwork as a major part of the module requires students to work together in teams and develop web-based software , this benefits students practical skills and teamwork while enhancing the students capability of managing software projects and their products.

COM1002 : Foundations of Computer Science

This module focuses on mathematical concepts and techniques that are significant to understanding the foundations of computing , this module also prepares you for future computing modules.

COM1003 : java Programming

This module teaches computer programming using the programming language Java , this module does not require you to have any prior knowledge of using Java, but you are expected to understand and write clear , readable and maintainable programs in java by the end of you classes

COM1004: Machines and intelligence

This module delves into the world of Artificial Intelligence , and introduces key points as well as problems in the field , it also compares the way computers understand things to how humans understand things and how they differ , students will also be provided with a historical overview of the subject .

1005: Devices and Network

This module provides students with information on how computer logic works and how computers understand information through logic gates , sequential logic while also explaining concepts like binary systems and how they operate and execute actions.

1006: Web and Internet technology

This module focuses on front end development and of interactive websites and tackles the practical use of HTML , CSS and javascript which are the programming languages mainly used for web development
